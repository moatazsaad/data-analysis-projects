### **Data Preparation and Index Tracking Strategy**  

1. **Initialize DataFrames:**  
   - Start with **const_prices** (stock prices with date index) and **DJI table**.  

2. **Merge Data:**  
   - Extract **closing prices** from the DJI table.  
   - Concatenate with **const_prices** to create a single DataFrame (**df**) containing stock prices and DJI values.  

3. **Enhancing Data:**  
   - Create **more_info** from **const** (includes ticker and company details).  
   - Normalize stock prices to ensure they all start from the same scale.  

4. **Calculating Returns & Risk:**  
   - Compute **daily returns** using `df.pct_change()`, creating the **ret DataFrame**.  
   - Generate a **summary** of returns and risk metrics.  
   - Merge this summary with **more_info** (setting the company as the index).  

5. **Data Cleaning & Visualization:**  
   - Create **summary2** by dropping DJI-related rows (to remove NaNs).  
   - Plot **high-return and high-risk** stocks.  
   - Create a scatter plot to analyze stock correlations.  

6. **Training and Forward Testing:**  
   - Train models on **2017–2018 data** and forward test on **2019**.  
   - Extract **2017–2018 data** from the original **df** (creating **df2**).  
   - Normalize **df2** (`norm2`).  
   - Compute **returns** for this period (`ret2`).  

7. **Tracking Error & Portfolio Construction:**  
   - Calculate **daily return differences** between each stock and DJI.  
   - Compute **annualized tracking error (TE)** to evaluate deviations.  
   - Identify **10 stocks with the lowest tracking error** to construct a tracking portfolio.  

8. **Building the Tracking Portfolio:**  
   - Create a **portfolio of 10 selected stocks** for index tracking.  
   - Normalize stock prices in **tracking_stocks** (`norm_track`).  
   - Remove **DJI** from **tracking_stocks**.  
   - Assign **weights** to each stock and compute portfolio returns.  
   - Implement functions for **portfolio returns** and **tracking error calculations**.
