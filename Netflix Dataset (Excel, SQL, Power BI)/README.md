# Netflix Shows Data Cleaning, Transformation & Analysis

**Tools:** Microsoft Excel, MySQL Workbench, Power BI

- Imported and cleaned large Netflix datasets using Excel; applied functions like TRIM, TEXT TO COLUMNS, and IF statements to normalize and prepare multi-valued columns (cast, directors, countries, categories).

- Split complex columns into multiple discrete fields and removed inconsistencies for relational database import.

- Imported datasets into MySQL Workbench, handled encoding issues, and optimized data structure by pivoting wide tables into normalized two-column formats (e.g., show_id with single attribute values).

- Validated data integrity and completeness post-import via SQL queries and row counts.

- Connected cleaned data to Power BI to build interactive dashboards with key metrics such as movie durations, release years, and show types.

- Iteratively refined data preparation and visualizations to support insightful business analysis.
